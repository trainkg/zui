总体信息

浏览器兼容性 IE8 以上

基于B-JUI 制作, B-JUI 非MVC结构，修改起来有点难度。

主要采用AMD规范将框架整合进来，由于B-JUI有一些全局的设定，暂时把B-JUI全局设定保留，看是否以后能融合良好

也可以采用增强或者修改方式进行

总体思想想把backbone MVC模式整合进来, 使用ractjs 增强，最大限度发挥出来响应式标签的能效

欢迎参与，要求能有独立书写jquery插件

大概介绍

框架总体采用 AMD 规范 requirejs, MVC框架 backbone , bootstrap jquery, 采用相应设计方案

目前有简单的组件生命周期监听处理，主要负责响应式的UI处理，这一块 reactjs 有良好的支持(日后看是否可以替换)

目前支持的插件有如下

  • navTab
  • 对话框
  • 消息提示
  • 页面布局
  • icheck
  • niceValidate
  • 简单日期选择器
  • 微调器
  • 右键菜单
  • bootstrap select
  • 预加载插件

动画处理

动画处理的基本原则是尽量采用CSS3动画定义, 在无CSS3支持的浏览器中降低体验的处理方案
